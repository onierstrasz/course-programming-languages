
These are the examples for the Programming Languages course held at the University of Bern, Switzerland.

http://scg.unibe.ch/teaching/pl

These examples are available from the following git repository:

	git clone git://scg.unibe.ch/lectures-pl-examples

This project contains examples for programs in the following languages:

	Haskell
	Prolog
	Postscript
	eToys
	AGG

---

There are also some C examples (not currently taught in this course). The C++ examples have moved to P2.

For Piccola examples, see the Piccola web page.

	http://scg.unibe.ch/research/piccola

Most of the programs are provided with tests. See the Makefiles in the individual directories.
